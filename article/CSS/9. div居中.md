# div 居中

让 `div` 内的元素水平垂直居中。

我们先给一段 `HTML` 代码，设置两个元素的嵌套。

```html
<div class="box">
  <div class="content">内部元素</div>
</div>
```

接下来，我们就都使用该元素来做操作。

## flex 居中

首先，当然要说一下，我们最常见的 [`flex` 布局]()的居中了。

我们来看看 `flex` 怎么居中。

```css
.box {
  width: 100px;
  height: 100px;
  background-color: #efcca1;
  display: flex;
  justify-content: center;
  align-items: center;
}
.content {
  width: 50px;
  height: 50px;
  background-color: #1890ff;
}
```

```html
<div class="box">
  <div class="content"></div>
</div>
```

我们就可以看到结果了。
